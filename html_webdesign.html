A brief introduction:

I would like to present three very different web design projects. The first one I built from scratch as a way to practice what I'd learned. The second one was created by me and my co-worker (and collegue) Jacek, and it was actually running online for over a year. And last but not least - the third project, a proffesional one, which I supervised and tested. It's is real so be carefull with clicking :).

First project:

When I first started learning HTML & CSS, I felt a natural need to build my own web page. So I gave it a try.
A tutor I found online said building a web page is like handling a window shutter - you're pulling from one side and the other side is moving up. Never before had I dealt with something equally satisfactory and devastating! But nevertheless I understood two things - I want to continue my IT adventure... but it probably won't be front-end...
This project looks both over-loaded (as I wanted to implement every new concept I'd learned) and abandoned (as I abandoned it at a certain stage and never continued).
